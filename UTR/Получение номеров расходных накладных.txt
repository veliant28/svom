Отримання номерів видаткових накладних
Для отримання видаткових накладних, пов'язаних із замовленням, необхідно виконати наступний запит:

GET https://order24-api.utr.ua/api/accounting_numbers_by_order/{order_code}
де:
{order_code} - номер замовлення (Наприклад: УЯТ555556)
додавши при цьому в headers

Authorization: Bearer eyJ0eXAiOiJKV1Q...


Де в Authorization після ключового слова Bearer потрібно передати отриманий при проходженні аутентификації token

В результаті ви отримаєте одну з наступних відповідей:

Помилка аутентифікації
Можливі відповіді API при помилці аутентификації приведено у розділі Помилка аутентификації

Список видаткових накладних
Status Code: 200 OK
Body:
[
    {
        "documentNumber": "БЮТ576594",
        "date": "2021-05-12T11:59:26",
        "accountingNumber": "75195"
    },
    ...
   ]
}
Где:
documentNumber - номер видаткової накладної
date - дата видаткової накладної
accountingNumber - електроний номер

Отримання номера замовлення
Для отримання замовлення, пов'язаного з видатковою накладною, виконати наступний запит:

GET https://order24-api.utr.ua/api/order_by_accounting_number/{number}
де:
{number} - електроний номер накладної (Наприклад: 75196)
додавши при цьому в headers

Authorization: Bearer eyJ0eXAiOiJKV1Q...


де в Authorization після ключового слова Bearer потрібно передати отриманий при проходженні аутентификації token

В результаті ви отримуєте одну з наступних відповідей:

Помилка аутентифікації
Можливі відповіді API при помилці аутентификації приведено у розділі Помилка аутентификації

Замовлення покупця
Status Code: 200 OK
Body:
[
    {
        "documentNumber": "УЯТ576594",
        "date": "2021-05-12T11:59:26",
    },
    ...
   ]
}
Де:
documentNumber - номер замовлення
date - дата замовлення
