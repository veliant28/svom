Довідник районів областей населених пунктів
Метод getSettlementCountryRegion працює в моделі AddressGeneral.  За допомогою цього методу можна завантажити довідник районів областей населених пунктів, аналогічний даним методу getSettlements.

Текст запиту
Параметр	Тип	Опис
AreaRef*	string[36]	Ідентифікатор (REF) області
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AddressGeneral",
   "calledMethod": "getSettlementCountryRegion",
   "methodProperties": {
"AreaRef" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
