Види часових інтервалів
Метод getTimeIntervals працює в моделі Common і використовується для отримання списку часових інтервалів (для замовлення послуги «Часові інтервали»).

 Довідник потрібно завантажувати один раз на місяць для оновлення даних.

Текст запиту
Параметр	Тип	Опис
RecipientCityRef*	string[36]	Ідентифікатор (REF) міста, для якого будуть отримані часові інтервали
DateTime	string[36]	Дата, за яку будуть отримані часові інтервали (необов'язковий параметр). Стандартне налаштування дорівнює поточній даті, формат: dd.mm.yyyy
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "CommonGeneral",
   "calledMethod": "getTimeIntervals",
   "methodProperties": {
"RecipientCityRef" : "00000000-0000-0000-0000-000000000000",
"DateTime" : "09-18-16"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.

Тіло відповіді на запит
Параметр	Тип	Опис
Number	string[50]	Ідентифікатор
Start	string[36]	Час початку інтервалу
End	string[36]	Час закінчення інтервалу
