Онлайн-пошук у довіднику населених пунктів
Метод searchSettlements працює в моделі Address. Цей метод необхідний для ОНЛАЙН-ПОШУКУ населених пунктів. Завдяки йому не потрібно зберігати довідники та здійснювати їхнє оновлення на стороні клієнта.

Текст запиту
Параметр	Тип	Опис
CityName*	string[36]	Назва населеного пункту або індекс населеного пункту
Limit*	string[36]	Кількість записів на сторінці. Працює разом із параметром Page
Page*	string[36]	Сторінка, працює разом із параметром Limit
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AddressGeneral",
   "calledMethod": "searchSettlements",
   "methodProperties": {
"CityName" : "київ",
"Limit" : "50",
"Page" : "2"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
