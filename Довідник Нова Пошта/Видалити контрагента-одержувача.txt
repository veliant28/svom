Видалити контрагента-одержувача
Метод delete працює в моделі Counterparty і використовується для видалення контрагента-отримувача. Для видалення контрагента-відправника необхідно звертатися до вашого менеджера.

Текст запиту
Параметр	Тип	Опис
Ref*	string[36]	Ідентифікатор (REF) контрагента
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "CounterpartyGeneral",
   "calledMethod": "delete",
   "methodProperties": {
"Ref" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
