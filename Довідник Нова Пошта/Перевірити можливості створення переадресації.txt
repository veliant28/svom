Перевірити можливості створення переадресації
Метод checkPossibilityForRedirecting працює в моделі AdditionalServiceGeneral і дозволяє перевірити можливість створення заявки на переадресацію відправлення.

Метод доступний лише клієнтам відправників.

У разі успіху повертаються необхідні параметри створення заявки.

Текст запиту
Параметр	Тип	Опис
Number*	string[36]	Номер документа
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AdditionalServiceGeneral",
   "calledMethod": "checkPossibilityForRedirecting",
   "methodProperties": {
"Number" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
