Довідник географічних областей України
Метод getAreas працює в моделі Address. Цей метод завантажує довідник географічних областей України, де є відділення компанії «Нова пошта».

Рекомендовано оновлювати довідник щомісяця.

Текст запиту
Параметр	Тип	Опис
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AddressGeneral",
   "calledMethod": "getAreas",
   "methodProperties": {   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
