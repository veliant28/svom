Отримати список підтипів причини повернення
Метод getReturnReasonsSubtypes працює в моделі AdditionalService і дозволяє отримати список підтипів доступних причин повернення.

Текст запиту
Параметр	Тип	Опис
ReasonRef*	string[36]	Ідентифікатор (REF) причини зі списку причин повернення
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AdditionalServiceGeneral",
   "calledMethod": "getReturnReasonsSubtypes",
   "methodProperties": {
"ReasonRef" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
