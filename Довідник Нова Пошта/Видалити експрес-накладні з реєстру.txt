Видалити експрес-накладні з реєстру
Метод removeDocuments працює в моделі ScanSheetGeneral і використовується для видалення експрес-накладних (ЕН) із реєстру в інформаційній системі «Нова пошта».

Текст запиту
Параметр	Тип	Опис
DocumentRefs*	array[36]	Масив ідентифікаторів документів, які необхідно видалити
Ref	string[36]	Ідентифікатор (REF) реєстру
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",

   "calledMethod": "removeDocuments",
   "methodProperties": {
"DocumentRefs" : [
"00000000-0000-0000-0000-000000000000",
"00000000-0000-0000-0000-000000000000"
],
"Ref" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.

Тіло відповіді на запит
Параметр	Тип	Опис
DocumentRefs	object	Відображає результат операції видалення документів з реєстру в інформаційній системі. Цей об'єкт може містити два основних поля:
Errors: Масив, що містить опис помилок, які виникли під час спроби видалення документів з реєстру. Якщо цей масив порожній, це означає, що помилок не виявлено, і всі заплановані до видалення документи були оброблені успішно.
Success: Масив об'єктів, кожен з яких представляє документ, що був успішно видалений з реєстру. Кожен об'єкт у цьому масиві містить такі атрибути:
Ref: Унікальний ідентифікатор (REF) документа, який було видалено
Number: Номер ІД
Document: Ідентифікатор реєстру, з якого було видалено документ
