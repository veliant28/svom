Видалити адресу контрагента (відправник / одержувач)
Метод delete працює в моделі Address. Цей метод необхідний для видалення адреси контрагента (відправника чи отримувача).

Редагувати дані контрагента можна лише з моменту його створення та до моменту створення інтернет-документа (ІД) із цим контрагентом.

Текст запиту
Параметр	Тип	Опис
Ref*	string[36]	Ідентифікатор (REF) адреси
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AddressGeneral",
   "calledMethod": "delete",
   "methodProperties": {
"Ref" : "00000000-0000-0000-0000-000000000000"
   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.
