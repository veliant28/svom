Отримати список причин повернення
Метод getReturnReasons працює в моделі AdditionalService і дозволяє отримати список доступних причин повернення.

Текст запиту
Параметр	Тип	Опис
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AdditionalServiceGeneral",
   "calledMethod": "getReturnReasons",
   "methodProperties": {   }
}

                                
                                  





                                
Відповідь 200
Завжди повертає код 200 навіть при логічних помилках.

Тіло відповіді на запит
Параметр	Тип	Опис
Ref	string[36]	Ідентифікатор (REF) причини
Description	string[100]	Опис причини українською мовою
APPLICATION/JSON
TEXT/XML
