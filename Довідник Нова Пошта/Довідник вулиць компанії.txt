Довідник вулиць компанії
Метод getStreet працює в моделі Address. Цей метод завантажує довідник вулиць в межах населених пунктів України, куди здійснює доставку компанія «Нова пошта». 

Довідник використовується під час створення відправлень з типом доставки від/до адреси клієнта. Щоб знайти конкретний проспект чи вулицю, потрібно в запит додати параметр FindByString (пошук за рядками) і задати назву у його значенні.

Текст запиту
Параметр	Тип	Опис
CityRef*	string[36]	Ідентифікатор (REF) міста, дані з Довідника міст компанії
FindByString*	string[36]	Пошук за рядками
Page	string[36]	Сторінка (до 500 записів на сторінці)
Limit	string[36]	Кількість записів на сторінці. Працює разом із параметром Page
* Позначення обов’язкового параметру
APPLICATION/JSON
TEXT/XML
                        

 
{
   "apiKey": "[ВАШ КЛЮЧ]",
   "modelName": "AddressGeneral",
   "calledMethod": "getStreet",
   "methodProperties": {
"CityRef" : "00000000-0000-0000-0000-000000000000",
"FindByString" : "Київ",
"Page" : "1",
"Limit" : ""
   }
}
